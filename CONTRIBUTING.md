# Contributing

Thanks for improving this documentation repository.

## Contribution Rules

- Use official sources first.
- Do not add real API keys, tokens, credentials, screenshots containing secrets, or private URLs.
- Mark hardware requirements as sourced, calculated, or estimated.
- Do not add unsupported benchmark claims.
- Keep examples copy-pasteable with placeholders only.
- For Claude Code backend replacement docs, distinguish Anthropic-compatible `/v1/messages` endpoints from OpenAI-compatible `/v1/chat/completions` endpoints.
- Update `docs/sources.md` and `data/sources.json` when adding new factual sources.
- Update translations when changing setup steps in `README.md`.

## Pull Request Checklist

- [ ] Markdown renders correctly on GitHub.
- [ ] Links are valid.
- [ ] Secrets are placeholders only.
- [ ] Gemma VRAM numbers are labeled as sourced, calculated, or estimated.
- [ ] New repositories have a trust level: official, curated, or community.
